Main window

Main window of the program consists of the following interface elements:

  • Main menu
  • Toolbar
  • Some dockable panels
  • Main table
  • Status bar

Main menu

Refer to Main menu section to get detailed menu command description.

Toolbar

Toolbar allows quick execution of frequently used commands. You can show or hide the toolbar with View | Toolbar main menu item. You can also drag and dock the toolbar with the mouse.

Dockable panels

Dockable panels, like the toolbar, can be dragged and docked with the mouse, and toggled using corresponding View menu item.
Current source entry Displays read-only text of the source component property or constant value.
Current translated entry Is a editor for entering translated component property or constant value.
Entry properties Allows to change flags for all selected entries at once.
    Untranslated Indicates that there's still no translated value for the entry. Such entries are not saved with the translation. This flag is automatically cleared when you change the translated value.
    Autotranslated Indicates that the translation was obtained using Translation Repository and hence you should check the value for accuracy. This flag doesn't affect way the entry translation is saved.
Bookmarks Displays list of bookmarks you can navigate to with a double-click or Enter key.

Main table

The main table consists of the four columns:
Component/Property/Constant Displays form, component property or constant name.
ID Displays component property ID.
Default value Displays original (source) value of a property or a constant.
Translated value Displays translated value of a property or a constant. To edit it you use Current translated entry dockable panel.

Status bar

Status bar shows hints and statistical information about the current translation and Translation Repository:
Comp Total number of component entries.
Prop Number of component property entries total/untranslated.
Const Number of constant entries total/untranslated.
Repository Number of Translation Repository entries.
