Installation

  1. Unzip all the files into a directory, eg. C:\Program Files\DKLang.
  2. Place a shortcut to the main application, DKTranEd.exe, onto your desktop and/or Start menu.
  3. You may want to associate language files to be opened with the program. You achieve this by double-clicking on a language source file (.dklang) or on a translation file (.lng) in Explorer, clicking Other program... and then selecting DKTranEd.exe as default application for these types of files. Translation Editor recognizes both the file types and places them in the appropriate fields in the Open files dialog.
