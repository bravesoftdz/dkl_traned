Open language files dialog

Open language files dialog is used to specify source and translation files to work with.

It appears automatically on program start letting you to select files with no additional manipulations.

To display the dialog manually use File | New or Open... main menu item or Ctrl+N or Ctrl+O shortcut keys.

Controls

Language source file Specifies a language source file (with .dklang extension) translation is based on. You may click Browse... button to browse for file.
Use the following translation file... Specifies an optional translation file from which to display the values instead of ones of the language source file. This might be useful when you make a translation into a language which is more close to language of another translation than to language of the source. You may click Browse... button to browse for file.
Create new translation When checked, a new empty translation will be created when you pressed OK. Also Translation properties dialog will be displayed to let you specify new translation settings.
Open an existing translation file When checked, you should specify a translation file to open and edit. You may click Browse... button to browse for file.
